# Hangman Game

A simple text-based Hangman game built with Python. Guess the hidden word one letter at a time before using all 6 chances.

## Features

* 5 predefined words
* Random word selection
* 6 incorrect guesses allowed
* Win/Lose conditions

## Requirements

* Python 3.x

## Run

```bash
python hangman.py
```

## Word List

apple, mango, grape, peach, melon
